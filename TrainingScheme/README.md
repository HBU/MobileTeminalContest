# 技能培训
### 新队员应具备基础知识：【iOS：C，C++】；【Android：java】
1. 在GitHub注册账号，建立自己的代码库。
2. 学习使用Git，把学习过程中积累的代码上传到代码库
3. 根据培训计划，完成每周进度，参与周例会。
4. 检验编写代码能力，准备组队参赛。
5. 组队原则：三人一组（大致分工：项目管理、UI设计、编码实现）
